# Modern React Template

## Install 
```bash
yarn
```

## Start
```bash
yarn start
# or
yarn dev
```

## Build
```bash
yarn build
```

## Miscelaneous scripts
```bash
yarn lint # Lint all files using eslint
yarn lint:fix # Fix all files using lint
yarn test # Run Jest
yarn check # Run All checks (eslint, jest, typescript)
```
